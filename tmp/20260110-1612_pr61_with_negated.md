## Part 2: Manual Delimiter Sizing Commands

Implements `\big`, `\Big`, `\bigg`, `\Bigg` and their variants for manual control of delimiter sizes.

### Supported Commands

| Command Family | Size | Multiplier |
|----------------|------|------------|
| `\big`, `\bigl`, `\bigr`, `\bigm` | Slightly larger | 1.0x font size |
| `\Big`, `\Bigl`, `\Bigr`, `\Bigm` | Larger | 1.4x font size |
| `\bigg`, `\biggl`, `\biggr`, `\biggm` | Even larger | 1.8x font size |
| `\Bigg`, `\Biggl`, `\Biggr`, `\Biggm` | Largest | 2.2x font size |
