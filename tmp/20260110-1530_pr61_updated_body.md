## Part 2: Manual Delimiter Sizing Commands

Implements `\big`, `\Big`, `\bigg`, `\Bigg` and their variants for manual control of delimiter sizes.

### Implementation
- Add `delimiterHeight` property to `MTInner` class for explicit size control
- Add `delimiterSizeCommands` dictionary with size multipliers based on standard TeX
- Parse sizing commands and create `MTInner` atoms with explicit delimiter height
- Modify `MTTypesetter.makeLeftRight()` to use explicit height when provided

### Supported Commands

| Command Family | Size | Multiplier |
|----------------|------|------------|
| `\big`, `\bigl`, `\bigr`, `\bigm` | Slightly larger | 1.0x font size |
| `\Big`, `\Bigl`, `\Bigr`, `\Bigm` | Larger | 1.4x font size |
| `\bigg`, `\biggl`, `\biggr`, `\biggm` | Even larger | 1.8x font size |
| `\Bigg`, `\Biggl`, `\Biggr`, `\Biggm` | Largest | 2.2x font size |

### Examples
```latex
\Bigg( \bigg( \Big( \big( x \big) \Big) \bigg) \Bigg)  % nested, outer=larger
f\big(g(x)\big) = \Big(\sum_{i=1}^n x_i\Big)           % mixed expression
```
